## Colour Coding System (for manual analysis)

When highlighting intake forms manually:
- 🔴 Red — Pain language (PAIN)
- 🟢 Green — Desired outcomes (DESIRE)
- 🟡 Yellow — Objections (OBJECTION)
- 🔵 Blue — Failed solutions (FAIL)
- 🟠 Orange — Trigger events (TRIGGER)
- 🟣 Purple — Functional limitations (LIMIT)

## Common Patterns in Healthcare Intake Forms

**Pattern: The Long Sufferer**
Language: "I've had this for [5+] years"
Awareness: Problem Aware, high scepticism
Copy implication: Acknowledge failure of previous treatments before introducing mechanism

**Pattern: The Acute Crisis**
Language: "It just started 2 weeks ago", "I can't work like this"
Awareness: Problem Aware, high urgency
Copy implication: Lead with availability (same-day), not education

**Pattern: The Researcher**
Language: "I've read about spinal decompression", "My friend said..."
Awareness: Solution Aware
Copy implication: Comparison and differentiation messaging

**Pattern: The Reluctant Returner**
Language: "I've tried chiro before but it didn't last"
Awareness: Product Aware, sceptical
Copy implication: Address why previous chiro failed; introduce new mechanism or approach
